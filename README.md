#BroadcastUnlockApp

App Android que utiliza un BroadcastReceiver para detectar el evento de desbloqueo de pantalla del dispositivo.

Cuando el usuario desbloquea el teléfono:
- Se muestra un mensaje (Toast)
- Se registra un evento en Logcat
- Se abre la aplicación de llamadas con un número predefinido
